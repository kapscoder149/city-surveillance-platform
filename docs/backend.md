# Backend Architecture

## Overview

The backend provides the operational services supporting surveillance management, infrastructure monitoring, and asset lifecycle management.

The application exposes REST APIs consumed by the frontend while coordinating monitoring services and operational workflows.

---

## Technology Stack

- Python
- FastAPI
- MariaDB
- Docker
- Docker Compose
- FFmpeg
- MediaMTX

---

## Core Services

- Asset Management
- Health Monitoring
- Camera Monitoring
- Evidence Management
- Playback Services
- Configuration Services

---

## Engineering Principles

- Layered architecture
- API-first development
- Modular service organization
- Configuration-driven design
- Production-ready deployment
