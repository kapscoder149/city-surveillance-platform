# Frontend Architecture

## Overview

The frontend was designed as a modular React application with an emphasis on scalability, maintainability, and operator usability.

The application follows a feature-first organization where each domain owns its components, state, services, and pages.

---

## Technology Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- TanStack Query
- React Router
- React Leaflet
- Radix UI
- shadcn/ui

---

## Design Principles

- Feature-first architecture
- Reusable component library
- Centralized state management
- Type-safe development
- Responsive operational dashboard
- Consistent design system
