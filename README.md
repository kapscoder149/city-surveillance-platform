# Project Overview

This project showcases the design and architecture of a production-inspired Surveillance Operations Platform built to monitor distributed edge devices and CCTV infrastructure through a centralized operations center.

The platform focuses on providing operators with a unified interface for infrastructure monitoring, surveillance management, evidence collection, health monitoring, GIS visualization, and operational workflows.

The project demonstrates software engineering practices including scalable architecture, modular frontend design, backend service organization, and AI-assisted development workflows.
